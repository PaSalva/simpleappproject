# Como ejecutar la función

## Requisitos
- Añadir SA y guardar en la carpeta con el nombre de `sa.json`

## Desarrollo local
- Run ```npm i && node index.js```
- Abrir navegador con localhost:5000

Listado de endpoints:
- /speech
